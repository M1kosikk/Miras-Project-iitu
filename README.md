# Miras-Project-iitu
Telegram chatbot about Directory for freshmen

Telegram-бот для помощи первокурсникам.

Возможности
- поиск кабинетов
- информация о WiFi
- расписание
- библиотека
- общежитие
- контакты

Технологии
- Visual Studio Code
- Python
- BotFather
- pyTelegramBotAPI

Запуск проекта

Установка библиотеки:

pip install pyTelegramBotAPI

Запуск:

python bot.py
